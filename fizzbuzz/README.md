# FizzBuzz

Display the numbers from 1 to 100, with a twist.

## Step 1
* If the number is a multiple of 3, display "Fizz"
* If the number is a multiple of 5, display "Buzz"
* If the number is a multiple of 3 and 5, display "FizzBuzz"

Samples

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
```
